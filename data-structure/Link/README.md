### JavaScript-Link

链表存储有序的元素集合，但不同于数组，链表中的元素在内存中并不是连续放置的。每个元素由一个存储元素本身的节点和一个指向下一个元素的引用（也就是指针或者链接）组成

--------

**<font color=#FF0000>使用环境:</font>** 顺序存放，顺序输出。方便删除，插入操作。对于计算机空间利用率大。  
要访问链表中的一个元素，需要从起点开始迭代列表直到找到所需的元素。
