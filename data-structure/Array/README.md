## JavaScript-Array
JavaScript的Array对象是用于构造函数组的全局对象，数组是类似于列表的高阶对象。  

数组对象是用来在单独的变量名中存储一系列的值。数组存储同一种数据类型的值。但是在JavaScript中，也可以在数组中存储不同类型的值。（但是为了规范，我们一般遵守最佳实践）
